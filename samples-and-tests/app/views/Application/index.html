chart module application

This demo application demonstrates the chart module by providing examples of all support chart types as well as all possible chart builder configuration options.

The code for this demo source consists of:
  • The view: index.html
  • The controller: Application.java
  • The routing rules: conf/routes

Chart as img with src set to a Base64-encoded image

All charts within this section are generated using:
  1. Instantiation of Chart instances and invocation of render() passing them as parameters — see controller action method index which is associated to path /
  2. Rendering of Chart instances passed by the controller in the view using fast tag chart.img
The following sections provide examples of the different types of charts supported by this module. For each type, two charts are displayed:
  1. A chart built using the default builder configuration and passing no additional attributes to the fast tag
  2. A chart built using all possible builder configuration settings and passing attribute title to the fast tag
