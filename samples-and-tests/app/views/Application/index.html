Chart module application

This demo application demonstrates the chart module by providing examples of all support chart types as well as all possible chart builder configuration options.

The code for this demo source consists of:
  • The view: index.html
  • The controller: Application.java
  • The routing rules: conf/routes

Chart as img with src set to a Base64-encoded image

All charts within this section are generated by:
  1. In the controller: creation of Chart instances and invocation of render() passing them as parameters — see controller action method index
  2. In the view: rendering of Chart instances using fast tag chart.img
The following sections provide examples of the different types of charts supported by this module. For each type, two charts are displayed:
  1. A chart built using the default builder configuration and passing it to the fast tag
  2. A chart built using all possible builder configuration settings and passing it along with HTML attribute title to the fast tag
